# Neural Network Training Chapter 4 Presentation Design

## 목적

`neural_net_train.ipynb`의 내용을 바탕으로 `DS_ppt_16_9_blue.pptx` 템플릿을 유지한 발표 자료를 만든다. 발표는 10분 기준이며, 전공자 청중을 대상으로 하되 수식 유도보다 개념 흐름과 학습 메커니즘 이해에 초점을 둔다.

## 확정된 조건

- 발표 제목: `신경망학습 chapter4`
- 발표 언어: 한국어
- 청중: 전공자 대상
- 발표 시간: 10분
- 슬라이드 구성 원칙: 한 섹터를 여러 장으로 나누는 것은 허용하며, 각 슬라이드는 상세한 유도보다 핵심 개념 전달을 우선한다.
- 템플릿 사용 원칙: `DS_ppt_16_9_blue.pptx`의 시각 스타일과 기본 톤을 유지하되, 텍스트와 이미지 콘텐츠는 발표 주제에 맞게 교체한다.
- 발표자 이름/소속: 소속은 사용자가 이미 수정해 둔 내용을 유지하며, 이름도 기존 템플릿에서 확인 후 필요한 범위만 반영한다.

## 추천 구성 방식

구성 방식은 `개념 흐름형`으로 확정한다. 노트북의 설명 순서를 크게 유지하면서 다음 개념으로 자연스럽게 이어지도록 구성한다.

핵심 흐름은 다음과 같다.

1. 신경망 학습의 목적 이해
2. 손실함수로 학습 목표 정의
3. 미니배치로 실제 학습 단위 설명
4. 수치 미분과 기울기로 업데이트 방향 이해
5. 경사하강법으로 실제 파라미터 이동 설명
6. 미니배치 학습 구현으로 chapter 4를 마무리

## 슬라이드 설계

예상 슬라이드 수는 12장 내외로 잡는다. 템플릿의 기존 6장만으로는 부족하므로 동일 양식의 슬라이드를 복제해 확장한다.

### 1. 표지

- 제목: `신경망학습 chapter4`
- 날짜, 소속, 발표자 정보는 기존 템플릿 구조를 유지하면서 주제에 맞게 갱신
- 표지 이미지는 신경망 학습 또는 gradient/optimization을 연상시키는 단순하고 전문적인 비주얼을 사용

### 2. 목차

- 손실함수
- 미니배치 학습
- 기울기와 경사하강법
- 학습 구현

### 3. 신경망 학습 개요

- 정답과 비교하며 가중치와 편향을 조정하는 과정이라는 점 설명
- 손실함수가 왜 필요한지 다음 슬라이드로 연결

### 4. 손실함수 소개

- 예측값과 정답 사이 차이를 하나의 숫자로 요약하는 기준
- 학습의 목표가 손실 최소화라는 점 제시

### 5. 오차제곱합

- 수식은 간단히 제시
- 큰 오차에 민감하다는 특징만 강조
- 분류 문제에서의 한계는 길게 다루지 않음

### 6. 교차 엔트로피 오차

- 정답 클래스 확률 중심 손실이라는 점 설명
- 분류 문제에서 자주 쓰이는 이유를 한두 문장으로 정리

### 7. 손실함수와 가중치 업데이트

- 손실값이 학습 방향을 만드는 신호라는 점 강조
- 업데이트 식 `w <- w - eta * dE/dw`를 제시
- 노트북의 `img/loss_guides_weight_update.svg`를 주요 그림으로 사용

### 8. 미니배치 학습

- 전체 데이터를 한 번에 쓰지 않고 일부 묶음으로 학습하는 이유 설명
- 계산 효율, 메모리 절약, 안정성 개선을 핵심 키워드로 정리
- 배치 샘플링 코드 일부를 짧게 요약해 제시

### 9. 수치 미분과 기울기

- 중앙차분 기반 수치 미분 개념 소개
- 편미분과 gradient 벡터의 의미를 연결
- 수식은 최소한으로 두고 “가장 가파르게 증가하는 방향”이라는 해석 중심

### 10. 경사하강법

- 학습률과 반복 업데이트 구조 설명
- 손실이 감소하는 방향으로 이동한다는 점 강조
- 가능하면 노트북에서 생성한 contour/path 시각 자료를 삽입

### 11. 기울기와 학습의 연결

- 단순 2변수 함수 예시와 실제 신경망 학습의 대응 관계 정리
- 현재 예측 -> 손실 계산 -> 기울기 계산 -> 파라미터 업데이트 순서를 한 장에서 정리

### 12. 미니배치 학습 구현

- `TwoLayerNet`, `MNIST`, loss 추적 코드 흐름을 개요 수준으로 설명
- 세부 코드 해설보다 “chapter 4에서 무엇을 구현했는가”를 요약

### 13. 마무리

- chapter 4 핵심 메시지 3개 정도로 정리
- 손실함수가 목표를 정의하고, 기울기가 방향을 정하며, 경사하강법이 실제 학습을 수행한다는 구조로 마무리

### 14. Q&A / Thank You

- 템플릿의 기존 마지막 슬라이드를 유지

## 시각 자료 원칙

- 기존 템플릿의 파란색 계열과 미니멀한 논문 발표 톤을 유지한다.
- 텍스트가 많은 슬라이드는 피하고, 각 슬라이드는 핵심 문장 2~4개 수준으로 압축한다.
- 노트북 내 제공 이미지인 `img/loss_guides_weight_update.svg`는 반드시 활용한다.
- 추가 이미지는 다음 우선순위로 반영한다.
  1. 노트북 실행으로 얻을 수 있는 gradient descent 관련 도식
  2. 내부에서 새로 생성한 단순 다이어그램 또는 아이콘형 그래픽
  3. 적절한 자료가 없을 경우 템플릿 톤에 맞춘 자체 제작 시각 요소

## 구현 범위

- 기존 PPT 템플릿을 직접 수정해 최종 발표용 파일을 만든다.
- 필요 시 슬라이드를 복제해 장수를 늘린다.
- 텍스트는 전면 한국어화한다.
- 이미지 배치는 템플릿의 균형을 유지하며, 과도한 장식보다 설명 보조 역할에 집중한다.

## 검증 기준

- 10분 발표 흐름으로 과밀하지 않아야 한다.
- 노트북의 주요 학습 개념이 빠지지 않아야 한다.
- 슬라이드별 메시지가 한눈에 읽혀야 한다.
- 템플릿의 스타일 일관성이 유지되어야 한다.
